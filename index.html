This project began as a CRUD module connected to a MongoDB database for managing animal rescue records. Algorithms & Data Structures – Course Planner (C++)

The Course Planner application reads course information from a CSV file and allows users to search for courses and view prerequisites. I enhanced this project by refining parsing logic, improving input validation, and implementing a map data structure to automatically maintain alphabetical order. These updates improved efficiency, readability, and reliability.
